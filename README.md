
Console Shop (C#)

Учебный консольный проект на C# моделирующий работу продуктового магазина:  
покупка товаров, оформление заказа, применение скидок, оплата и доставка.

Проект создан с упором на Чистую архитектуру, *SOLID* и Паттерны проектирования, а не на интерфейс.

Возможности

- Просмотр списка товаров
- Добавление товаров в заказ
- Подсчёт общей стоимости заказа
- Применение промокодов (скидок)
- Выбор способа оплаты (карта / наличные / онлайн)
- Управление состоянием заказа (создан → оплачен → доставлен)

 Архитектура проекта

Проект разделён на слои:

- Domain
  - `Product`, `Order`, `Customer`, `Buy`
- Business Logic
  - `OrderCalculator`
  - `DiscountFactory`
  - `PaymentFactory`
- Patterns
  - State (состояния заказа)
  - Factory (создание оплат и скидок)
- UI
  - `ConsoleMenu` (консольное взаимодействие)

Использованные паттерны проектирования

Factory
Используется для создания способов оплаты и промокодов/

Пример:
csharp
IPayment payment = PaymentFactory.Create(type);
IDiscountPromo promo = DiscountFactory.Create(code);
